How it works

Behind the demo: VerfiX Swiss Trust Infrastructure.

Document verification

VerfiX supports over 4,500 document types from 190+ countries. OCR recognition, security feature checks and biometric matching in real time.

Liveness detection

Active detection of deepfakes, replay attacks and synthetic voices. Continuously trained with current attack vectors.

AML screening

Real-time checks against SECO, UN, EU, OFAC and PEP register. Adverse media monitoring included.

Trust Engine™

All signals feed into a proprietary Trust Score (0–100). Clear decisions in under 3 seconds.
