Mono uppercase section kicker with a red dash — the signature VerfiX eyebrow above a heading.

```jsx
<Label>Trust Engine™</Label>
<h2>Every signal. One decision.</h2>
```

On dark sections pass a translucent white `color` (e.g. `"rgba(255,255,255,.35)"`). Set `dash={false}` for a bare mono caption.
