The core container. Hairline border, tight 8px radius, sits flush — no soft floating shadows by default.

```jsx
<Card accent hover>
  <IconChip icon="fas fa-layer-group" />
  <h4>Multi-Signal Fusion</h4>
  <p>Document, face, voice, device and AML in one trust score.</p>
</Card>
<Card dark>…</Card>
```

`accent` adds the red top line (feature cards). `dark` is the navy variant for emphasis cards inside light sections. `hover` enables the 2px lift used on interactive grids.
