Neutral keyword chip for capability lists under feature copy.

```jsx
<div style={{display:'flex',gap:'.22rem',flexWrap:'wrap'}}>
  <Tag>OCR</Tag><Tag>Liveness</Tag><Tag>PEP screening</Tag>
</div>
```

Always lay tags out in a flex row with `gap`. Keep them lowercase-ish technical terms; don't use them for status (use Badge).
