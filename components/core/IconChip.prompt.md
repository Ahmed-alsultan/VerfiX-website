The signature VerfiX feature glyph — a Font Awesome icon in a tinted red square. Used at the top of feature cards and beside list items.

```jsx
<IconChip icon="fas fa-shield-halved" />
<IconChip icon="fas fa-gauge-high" tone="solid" size="lg" />
```

Tones: `tint` (red on pink, default), `solid` (white on red — active/selected), `dark` (on navy sections), `plain` (neutral bordered). Always feed it a real Font Awesome icon — never an emoji.
