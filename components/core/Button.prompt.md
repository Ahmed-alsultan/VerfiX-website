Brand action button — red is the primary CTA, used once per view; navy/outline/ghost are secondary. Pair with Font Awesome icons.

```jsx
<Button variant="red" size="lg" icon="fas fa-calendar-check">Book a Demo</Button>
<Button variant="outline" size="lg">Explore Platform</Button>
<Button variant="ghost" iconRight="fas fa-arrow-right">Read docs</Button>
```

Variants: `red` (default CTA), `navy` (dark sections / secondary solid), `outline` (neutral secondary), `ghost` (tertiary, text-like), `white` (on navy backgrounds). Sizes: `sm` / `md` / `lg`. Hover lifts 1px; red gains a soft red shadow.
