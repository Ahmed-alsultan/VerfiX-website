Identity Verification for Individuals

VerfiX KYC: Identity Verification

VerfiX KYC enables regulated organisations to verify the identity of individuals digitally, in compliance with FINMA Circular 2016/7 and GwG Art. 3, with an immutable audit trail generated automatically for every verification event.

Executive Summary

Manual identity verification is slow, costly, inconsistent, and creates compliance risk. VerfiX KYC replaces manual document review, in-person identification, and fragmented verification tools with a single API that performs document verification, biometric matching, liveness detection, and audit trail generation in under three seconds.

All processing is conducted in Switzerland. The resulting audit trail meets FINMA inspection requirements without additional configuration.

Core Capabilities

Capability Technical Detail Compliance Value
Document Verification 4,500+ document types. 190+ countries. MRZ validation, security feature checks, tamper detection, chip data reading. Satisfies GwG Art. 3 identification requirement. Meets FINMA Circular 2016/7 document standards.
OCR & MRZ Extraction Extracts name, date of birth, document number, expiry, nationality from Machine Readable Zone with >99% accuracy. Enables structured data capture for compliance records without manual transcription errors.
NFC Chip Reading Reads and validates biometric data from ICAO 9303-compliant ePassport chips. Verifies digital signature from issuing authority. Highest assurance level identity verification. Chip-authenticated data cannot be physically counterfeited.
Face Verification ISO 30107-3 biometric matching. Compares live facial capture against document photograph. Configurable match threshold. Required by FINMA Circular 2016/7 for online identification. Match score retained in audit trail.
Liveness Detection Active (challenge-response) and passive (video analysis) liveness detection. Anti-spoofing against photo, video, and 3D mask attacks. Required by FINMA Circular 2016/7. Prevents presentation attacks. Liveness result retained in audit trail.
Voice Biometrics Speaker verification for telephone banking authentication. Swiss German, French, Italian, Romansh support. Enables FINMA-compliant telephone banking identification for cantonal and retail banks.
Audit Trail Immutable, timestamped, cryptographically signed record per verification. Includes all signals, scores, and outcomes. FINMA inspection ready. Supports 10-year GwG retention requirement. Exportable on demand.

Supported Document Types

VerfiX KYC supports more than 4,500 document types from 190+ countries, including:

  • ›Passports (ICAO 9303-compliant, including NFC chip reading for ePassports)
  • ›National identity cards (including European ID cards with NFC chips)
  • ›Driving licences (including EU driving licences and major non-EU formats)
  • ›Residence permits and visas
  • ›Swiss-specific documents: Swiss passport, Swiss identity card, Swiss driving licence, Swiss residence permit (Ausländerausweis)

Workflow

⬡ Diagram: KYC Verification Flow — (1) Customer captures document front/back → (2) OCR + MRZ extraction → (3) Security feature validation → (4) NFC chip read (if ePassport) → (5) Customer captures selfie → (6) Liveness detection → (7) Biometric face match → (8) Trust Engine risk score → (9) Decision (Approve / Review / Reject) → (10) Audit trail generated → (11) Webhook notification to organisation's system

Integration

VerfiX KYC is available via REST API. A verification is initiated with a single API call containing the document image(s) and selfie. The API returns a structured JSON response with all extracted data, verification results, fraud signals, and the audit record identifier.

Example request: POST https://api.verfix.ch/v1/verify/document

Response includes: document_type, country, extracted_fields, mrz_valid, security_features_valid, fraud_score, biometric_match_score, liveness_result, trust_score, decision, audit_id

Integration resources: REST API documentation, Postman collection, sandbox environment, technical support during integration.
