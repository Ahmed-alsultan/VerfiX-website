Part 2 — Activation Steps (in order)
1

Create Your Free HubSpot Account

5 minutes — one time only

  1. Go to app.hubspot.com → click "Get started free"
  2. Sign up with your hello@verfix.ch email
  3. Select "Marketing & Sales" as your use case
  4. Skip the team setup for now — you can add Manel and Petter later
  5. You land in the HubSpot dashboard — the free CRM is immediately active
The free tier includes: CRM contacts, live chat, forms, email tracking, and the tracking pixel. No credit card needed.
2

Get Your Hub ID — Replace in All 36 Pages

2 minutes

  1. In HubSpot: click your account name → Account & Billing
  2. Copy your Hub ID (a number like 12345678)
  3. Open the website ZIP, then run this command in your terminal:
# Replace YOUR_HUB_ID with your actual Hub ID in all 36 HTML files find . -name "*.html" -exec sed -i 's/YOUR_HUB_ID/12345678/g' {} + # Verify it worked: grep -r "YOUR_HUB_ID" . --include="*.html" | wc -l # Should return: 0

Or use VS Code: Edit → Replace in Files → search YOUR_HUB_ID, replace with your actual number.

What this activates: Visitor tracking on all 36 pages, contact identification when someone submits a form, and the live chat widget carrier script.
3

Activate Live Chat — Set Greeting Texts

10 minutes in HubSpot dashboard

  1. In HubSpot: Conversations → Inbox → Chat
  2. Click "Create chatflow" → choose "Live chat"
  3. Under Welcome message, set these three versions:

🇩🇪 German pages

"Fragen zur Compliance? Wir sind da."
Create a separate chatflow for DE pages, or use the default (most traffic is DE).

🇫🇷 French pages

"Des questions sur la conformité ? Nous sommes là."
Target: URLs containing /fr/

🇬🇧 English pages

"Compliance questions? We're here."
Target: URLs containing /en/

  1. Under Availability: set your timezone to Europe/Zurich
  2. Under Target: set "Show on all pages" (the code already controls language per page)
  3. Click Publish — chat widget goes live immediately after your Hub ID is in the code
Important: The chat widget loads via the HubSpot tracking script (Step 2). It will NOT appear until you have replaced YOUR_HUB_ID in the HTML and redeployed.
4

10 minutes

  3. Click Publish
  4. Click "Share" → "Embed" → copy the portalId and formGuid
// The HubSpot embed snippet looks like this: hbspt.forms.create({ portalId: "12345678", // ← this is YOUR_HUB_ID formId: "a1b2c3d4-e5f6-..." // ← this is YOUR_FORM_ID });

  2. Find the two placeholders near line ~200 and replace them:
var PORTAL_ID = 'YOUR_HUB_ID'; // → replace with e.g. '12345678' var FORM_ID = 'YOUR_FORM_ID'; // → replace with e.g. 'a1b2c3d4-e5f6-7890-abcd-ef1234567890'
Until these are replaced, contact.html shows a friendly fallback message with a mailto: link. No broken UI — visitors can always email hello@verfix.ch.
5

Connect Calendly → HubSpot (Demo Bookings → CRM)

Optional but strongly recommended

Every time someone books a demo via Calendly, you want that contact automatically created in HubSpot. Two options:

Option A — Native Integration (Easiest)

In HubSpot: App Marketplace → search "Calendly". The native integration syncs bookings as HubSpot contacts and meetings. Free on HubSpot Starter+.

Option B — Zapier (Free tier available)

zapier.com → New Zap → Trigger: "Calendly — New Invitee" → Action: "HubSpot — Create Contact". Free Zapier tier allows 100 tasks/month.

Recommended workflow once connected:
Calendly booking → HubSpot contact created → HubSpot sends automated confirmation email → Lead assigned in CRM → Ahmed gets Slack/email notification.
6

Set Region — EU Data Storage (Important for nFADP)

Must do at account creation

HubSpot offers EU data hosting (Frankfurt data centre). Since VerfiX is nFADP-compliant and Swiss-hosted, you should store CRM data in Europe.

  1. During account creation: select Europe as your data centre region
  2. If already created with US region: contact HubSpot support to migrate (it is possible)
  3. In the form embed script, confirm region: 'eu1' is set — it already is in the VerfiX code
  4. In HubSpot: Settings → Account Defaults → Data Privacy → enable GDPR settings (covers nFADP-aligned consent flows)
Do not use region 'na1' (US) for a Swiss B2B fintech platform. The code defaults to 'eu1' — verify this matches your HubSpot account's actual region.
7

Redeploy and Verify

Final step

  1. After replacing all placeholders, rezip and redeploy to Vercel
  2. Open verfix.ch in a private/incognito window
  3. After 5–10 seconds: HubSpot chat widget should appear bottom-right
  4. Go to verfix.ch/contact — the booking calendar (Calendly) should load above the fold, HubSpot form below
  5. Submit a test form — check HubSpot → Contacts for the new entry
  6. In HubSpot: Reports → Analytics Tools → Traffic Analytics → verify page views are being tracked
HubSpot chat icon position: The HubSpot chat widget (bottom-right) and the Calendly floating button (also bottom-right) may overlap. If they do: in HubSpot → Chat → Appearance → set widget position to bottom-left. The Calendly button stays bottom-right. No code change needed.
