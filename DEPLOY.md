# VerfiX Website — Deployment Guide

## Files in this package

| File | Purpose |
|------|---------|
| `index.html` | Main website — upload as your homepage |
| `og-image.jpg` | Social sharing image (1200×630) |
| `robots.txt` | SEO crawler instructions |
| `sitemap.xml` | Search engine sitemap |
| `.htaccess` | Apache server config (HTTPS redirect, headers, cache) |
| `_headers` | Netlify / Cloudflare Pages security headers |
| `_redirects` | Netlify route redirects |
| `vercel.json` | Vercel deployment config |

---

## Option A — Netlify (Recommended, free)

1. Go to [netlify.com](https://netlify.com) → New site → Deploy manually
2. Drag and drop **all files** into the upload area
3. Set custom domain: `verfix.swiss` in Site Settings → Domain Management
4. Enable HTTPS (automatic with Let's Encrypt)
5. Done ✓

Files needed: `index.html`, `og-image.jpg`, `robots.txt`, `sitemap.xml`, `_headers`, `_redirects`

---

## Option B — Vercel (Free)

1. Go to [vercel.com](https://vercel.com) → Add New → Deploy
2. Upload folder or connect GitHub repo
3. Add domain `verfix.swiss` in Project Settings → Domains
4. Done ✓

Files needed: `index.html`, `og-image.jpg`, `robots.txt`, `sitemap.xml`, `vercel.json`

---

## Option C — Cloudflare Pages (Free, Swiss CDN)

1. Go to Cloudflare Dashboard → Pages → Create a project
2. Upload files directly
3. Set custom domain `verfix.swiss`
4. Done ✓

Files needed: `index.html`, `og-image.jpg`, `robots.txt`, `sitemap.xml`, `_headers`, `_redirects`

---

## Option D — Traditional Hosting / cPanel

1. Upload all files to `public_html/` via FTP or File Manager
2. `.htaccess` handles HTTPS redirect automatically
3. Point DNS to your host

Files needed: ALL files including `.htaccess`

## Contact Form Setup

The contact form currently shows an alert on submit.
To make it functional, add one of these:

**Netlify Forms (easiest):**
Add `netlify` attribute to the form tag in index.html:
```html
<form netlify name="contact">
```

**Formspree (any host):**
```html
<form action="https://formspree.io/f/YOUR_ID" method="POST">
```
